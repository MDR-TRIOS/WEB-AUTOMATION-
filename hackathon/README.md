## 🛠️ Customization

### Change Sample Data
**In home.html:**
```javascript
const formData = {
    name: "Your Name",
    email: "your.email@example.com",
    // Change values here
};
```

**In form_autofill_helper.py:**
```python
SAMPLE_DATA = {
    'full_name': {'type': 'text', 'value': 'Your Name'},
    // Change values here
}
```

### Change Form URL
Update `FORM_URL` variable in both files

### Add New Fields
1. Add to `formData` object
2. Update modal preview
3. Add to Python `SAMPLE_DATA`
4. Test with new field

## 🐛 Troubleshooting

### Forms button not visible?
- Refresh browser
- Check browser console (F12)
- Verify home.html loaded correctly

### Python module errors?
```bash
pip install -r requirements_form_filler.txt
```

### Chrome not found?
- [Install Chrome](https://google.com/chrome)
- Or install Chromium: `pip install chromium`

### Form not autofilling?
- Use Selenium method for full automation
- Check logs for error details
- Ensure all required fields are being filled
